# Bruno API Collection for Sift Backend

## Folder Structure
```
sift/backend/bruno-collection
├── bruno.json
├── environments/
│   └── Local.bru
├── Auth/
│   ├── Register.bru
│   └── Login.bru
├── Jobs/
│   ├── Create Job.bru
│   ├── Get All Jobs.bru
│   ├── Get Job by ID.bru
│   └── Delete Job.bru
└── Candidates/
    ├── Upload CV.bru
    ├── Get Candidates.bru
    ├── Get Candidate by ID.bru
    ├── Reprocess Candidate CV.bru
    └── Delete Candidate.bru
```

## Usage Instructions

1. Install Bruno from https://www.usebruno.com/
2. Open the folder in Bruno
3. Start testing your API!

> Note: The token is automatically saved to the environment after login/register.
